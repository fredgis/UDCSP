# Foundry Golden Datasets

## Purpose
Small but schema-realistic evaluation inputs for classifier, translator, eligibility and safety tests. Larger generated counterparts are under data/synthetic/eval-datasets.

## How to regenerate
Use data/synthetic/generators/generate_eval_datasets.py for regenerated variants.

## How to validate
Check JSONL parseability and thresholds through foundry/evaluations scripts.
