# WCAG 2.1 AA checklist

- Perceivable: text alternatives, captions, adaptable layouts, contrast.
- Operable: keyboard access, focus order, no keyboard traps, enough time, skip links.
- Understandable: language attributes, plain errors, consistent navigation.
- Robust: semantic HTML, ARIA only when needed, screen-reader tested.

Mapped components: navigation, forms, status banner, language switcher, chat entry, document upload, error summary, notification panel.
