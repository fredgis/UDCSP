# Web i18n Catalogues

## Purpose
Source-of-truth ICU MessageFormat catalogues, translation pipeline, glossary and accessibility/localisation QA assets for agent-frontend hand-off.

## How to regenerate
Update messages/en.json first, run scripts/Translate-All.ps1, then human-review locale diffs.

## How to validate
Run scripts/Validate-Translations.ps1 to confirm every locale has the same keys as en.json.
