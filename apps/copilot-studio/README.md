# Copilot Studio Assets

## Purpose
Copilot Studio export scaffold for the citizen assistant bot, multilingual topics, Foundry connections, knowledge-source registrations and escalation rules.

## How to regenerate
Use scripts/Export-CopilotStudio.ps1 or Import-CopilotStudio.ps1 after tenant connection configuration.

## How to validate
Import into a non-production Copilot Studio environment and run topic tests in 12 languages.
