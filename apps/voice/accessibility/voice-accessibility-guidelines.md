# Voice accessibility guidelines

- Provide TTY/relay-service support for all country support numbers.
- Offer slow-speech mode from every greeting using voice phrase "speak slowly" or DTMF `9`.
- Allow repeat-on-demand from every prompt using phrase "repeat" or DTMF `*`.
- Keep menus to four options or fewer; announce keypad alternatives after voice choices.
- Support barge-in, silence timeouts, and human transfer without penalty.
- Never require speech: every top-level IVR path must have DTMF fallback.
- Read recording/transcription consent before capture and allow opt-out with human routing.
