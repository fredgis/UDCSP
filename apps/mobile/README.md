# UDCSP Mobile Shell

## Purpose
Expo/React Native citizen shell mirroring the primary web journeys for accessible mobile access.

## Dev setup
Use Node.js 20 and Expo CLI. Do not install during scaffold generation; run `npm install` once package feeds and i18n catalogues exist.

## Build
`npm run build` is a placeholder for EAS build profiles in `eas.json`.

## Test
`npm test` runs Jest unit tests for utility/component smoke coverage.

## Deploy
Use EAS profiles `dk`, `se`, and `no` to publish country-specific build channels.
